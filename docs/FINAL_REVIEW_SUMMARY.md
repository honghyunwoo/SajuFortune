## 📋 Executive Summary

### 프로젝트 개요
SajuFortune은 한국 전통 명리학(사주팔자)을 웹 기반으로 제공하는 서비스입니다. 사용자가 생년월일시를 입력하면 사주 분석, 격국 판별, 대운 계산, 십이운성 분석 등 전문적인 명리학 해석을 제공합니다.

### 1. 명리학 핵심 시스템 구현 (2025-10-02)

#### 격국 분석 시스템 (Geokguk Analysis)
**파일**: `shared/geokguk-analyzer.ts` (400 lines)

**구현 내용**:
- 8대 정격 자동 판별 (정관격, 편관격, 정재격, 편재격, 정인격, 편인격, 식신격, 상관격)
- 특수격 분석 (종격, 화격)
- 용신/희신/기신 자동 추출
- 격국 강도 점수화 (0-100)
- 상세 해석 (장점, 단점, 적합 직업, 주의사항)

**기술적 하이라이트**:
- TypeScript 타입 시스템으로 명리학 규칙 검증
- 순수 함수 패턴으로 테스트 용이성 확보
- 400줄의 복잡한 비즈니스 로직을 명확한 함수로 분리

**코드 예시**:
```typescript
export interface 격국결과 {
    격국명: 격국타입;
    격국종류: '정격' | '특수격' | '무격';
    격국강도: number; // 0-100
    용신: 오행타입;
    희신: 오행타입[];
    기신: 오행타입[];
    격국함의: string;
    상세해석: {
        장점: string[];
        단점: string[];
        적합직업: string[];
        주의사항: string[];
    };
}
```
