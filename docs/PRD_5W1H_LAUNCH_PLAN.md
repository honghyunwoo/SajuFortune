### 6️⃣ HOW (어떻게 실행하는가?)

#### Step-by-Step 실행 가이드

##### STEP 1: 데이터베이스 설정 (30분)
```bash
# 1. Neon (https://neon.tech) 가입
# 2. 새 프로젝트 생성: "sajufortune-prod"
# 3. Connection String 복사

# 4. 로컬 .env 업데이트
DATABASE_URL=postgresql://[user]:[password]@[host]/[database]?sslmode=require
```

##### STEP 2: Redis 설정 (20분)
```bash
# 1. Upstash (https://upstash.com) 가입
# 2. 새 Redis 데이터베이스 생성
# 3. REST URL & Token 복사

REDIS_URL=redis://default:[password]@[host]:6379
```

##### STEP 3: 배포 플랫폼 설정 (40분)
```bash
# Vercel 사용 시
# 1. https://vercel.com 가입 (GitHub 연동)
# 2. "New Project" → GitHub repo 선택
# 3. Environment Variables 설정:
#    - DATABASE_URL
#    - REDIS_URL
#    - SESSION_SECRET
#    - NODE_ENV=production
# 4. Deploy 클릭

# Railway 사용 시
# 1. https://railway.app 가입
# 2. "New Project" → "Deploy from GitHub"
# 3. 환경변수 동일하게 설정
```

##### STEP 4: 도메인 설정 (30분)
```bash
# 1. 도메인 구매 (가비아, 후이즈 등)
#    - sajufortune.kr (~15,000원/년)
#    - 또는 sajufortune.com (~15,000원/년)

# 2. DNS 설정
#    - Vercel: 제공되는 CNAME 레코드 추가
#    - Railway: 제공되는 A 레코드 추가

# 3. SSL 자동 적용 대기 (5-30분)
```

##### STEP 5: 광고 설정 (1시간)
```bash
# 1. Google AdSense 신청
#    - https://adsense.google.com
#    - 사이트 소유권 확인 (meta 태그 또는 ads.txt)
#
# ⚠️ 주의: AdSense 승인은 1-14일 소요
#    - 승인 전까지 광고 미노출
#    - 콘텐츠 충실히 채우기 (블로그 글 등)

# 2. 코드에 AdSense 컴포넌트 이미 있음
#    - client/src/components/ads/adsense-banner.tsx
#    - client/src/components/ads/adsense-inline.tsx
#
# 3. 퍼블리셔 ID만 업데이트
```

##### STEP 6: SEO 체크리스트
```bash
# 이미 구현된 것들 확인:
# ✅ client/src/components/seo-head.tsx
# ✅ 메타 태그 동적 생성
# ✅ OG 태그

# 추가 확인:
# □ public/robots.txt
# □ public/sitemap.xml
# □ Google Search Console 등록
# □ 네이버 서치어드바이저 등록
```
