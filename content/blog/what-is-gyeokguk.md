# 격국(格局)이란 무엇인가?

격국(格局)은 사주팔자의 **구조와 패턴**을 분석하여 그 사람의 운명적 특징을 파악하는 명리학의 핵심 개념입니다. 마치 건물의 설계도처럼, 격국은 개인의 인생 설계를 보여줍니다.

## 격국의 중요성

격국은 단순히 오행의 많고 적음을 세는 것이 아닙니다:

- **성격의 방향성**: 어떤 방식으로 세상을 대하는가
- **성공의 패턴**: 어떤 방식으로 목표를 달성하는가
- **인생의 과제**: 어떤 것을 극복해야 하는가

## 격국의 두 가지 큰 분류

### 1. 정격(正格) - 정상적인 구조
일간이 월지의 기운을 받아 **적절한 균형**을 이룬 경우입니다.

**8가지 정격**:
- 정관격(正官格)
- 편관격(偏官格) = 칠살격
- 정인격(正印格)
- 편인격(偏印格)
- 식신격(食神格)
- 상관격(傷官格)
- 정재격(正財格)
- 편재격(偏財格)

### 2. 외격(外格) - 특수한 구조
일반적인 패턴을 벗어난 **특별한 구조**입니다.

**주요 외격**:
- 종격(從格): 일간이 너무 약해 다른 오행을 따르는 경우
- 화격(化格): 천간이 합하여 다른 오행으로 변화하는 경우
- 전왕격(專旺格): 한 오행이 극도로 강한 경우

## 정격의 종류와 특징

### 1. 정관격(正官格)
월지에 정관이 있고 일간이 적당한 경우

**특징**:
- 성격: 질서와 원칙을 중시
- 적성: 공직, 관리직, 법조인
- 장점: 책임감, 성실함, 명예 중시
- 단점: 융통성 부족, 권위주의적

**예시**: 일간이 갑(甲)인데 월지가 유(酉, 금)인 경우

### 2. 편관격(偏官格) = 칠살격
월지에 편관이 있는 경우

**특징**:
- 성격: 강인하고 도전적
- 적성: 군인, 경찰, 운동선수
- 장점: 추진력, 결단력, 리더십
- 단점: 충동적, 공격적

**예시**: 일간이 갑(甲)인데 월지가 신(申, 금)인 경우

### 3. 정인격(正印格)
월지에 정인이 있는 경우

**특징**:
- 성격: 학구적이고 사려 깊음
- 적성: 교육자, 연구원, 예술가
- 장점: 지혜, 학습능력, 자비심
- 단점: 우유부단, 의존적

**예시**: 일간이 갑(甲)인데 월지가 해(亥, 수)인 경우

### 4. 식신격(食神格)
월지에 식신이 있는 경우

**특징**:
- 성격: 낙천적이고 창의적
- 적성: 요리사, 예술가, 사업가
- 장점: 재능, 복록, 여유
- 단점: 게으름, 안일함

**예시**: 일간이 갑(甲)인데 월지가 오(午, 화)인 경우

### 5. 상관격(傷官格)
월지에 상관이 있는 경우

**특징**:
- 성격: 창의적이고 표현력이 뛰어남
- 적성: 예술가, 연예인, 비평가
- 장점: 재능, 지혜, 표현력
- 단점: 비판적, 반항적

**예시**: 일간이 갑(甲)인데 월지가 사(巳, 화)인 경우

### 6. 정재격(正財格)
월지에 정재가 있는 경우

**특징**:
- 성격: 현실적이고 근면함
- 적성: 사업가, 회계사, 금융인
- 장점: 재물운, 성실함, 책임감
- 단점: 물질주의, 인색함

**예시**: 일간이 갑(甲)인데 월지가 진(辰, 토) 또는 술(戌, 토)인 경우

### 7. 편재격(偏財格)
월지에 편재가 있는 경우

**특징**:
- 성격: 활동적이고 사교적
- 적성: 영업, 무역, 투자
- 장점: 사교성, 활동력, 추진력
- 단점: 낭비, 경솔함

**예시**: 일간이 갑(甲)인데 월지가 축(丑, 토) 또는 미(未, 토)인 경우

## 격국 판단 방법

격국을 판단하는 기본 순서:

### 1단계: 월지 확인
월지(月支)에 무엇이 있는지 확인합니다.

**예시**: 갑목 일간, 유금 월지
- 유금은 갑목의 정관 → 정관격 가능성

### 2단계: 투출 확인
월지의 기운이 천간으로 **투출(透出)**되었는지 확인합니다.

**투출이란?** 지지의 기운이 천간에 나타나는 것
- 월지 유(酉, 금) + 년간/월간/시간에 경(庚) 또는 신(辛) → 투출됨

### 3단계: 일간 강약 판단
일간이 월령의 생지왕묘와 득령 여부를 확인합니다.

**득령**: 월지에서 일간이 왕성한 기운을 받는 것
- 봄(인묘진): 목이 득령
- 여름(사오미): 화가 득령
- 가을(신유술): 금이 득령
- 겨울(해자축): 수가 득령

### 4단계: 용신 찾기
**용신(用神)**: 사주를 조화롭게 만드는 핵심 글자

용신의 종류:
- **부족한 것을 채우는 용신**
- **과한 것을 제압하는 용신**
- **습하거나 조한 것을 조절하는 용신**

## 격국 활용 예시

### 사례 1: 정관격 사주
```
년주: 경오(庚午)
월주: 신유(辛酉) ← 유금 월지
일주: 갑인(甲寅) ← 갑목 일간
시주: 을해(乙亥)
```

**분석**:
- 일간: 갑목(甲木)
- 월지: 유금(酉金) - 갑목의 정관
- 월간: 신금(辛金) - 투출됨
- **결론**: 정관격

**특징**: 공직이나 관리직에 적합, 명예와 책임감 중시

### 사례 2: 식신격 사주
```
년주: 갑자(甲子)
월주: 병오(丙午) ← 오화 월지
일주: 갑진(甲辰) ← 갑목 일간
시주: 정사(丁巳)
```

**분석**:
- 일간: 갑목(甲木)
- 월지: 오화(午火) - 갑목의 식신
- 월간: 병화(丙火) - 투출됨
- **결론**: 식신격

**특징**: 창의적 재능, 예술이나 음식 관련 직업 적합

## 격국과 운명

격국이 정해지면:

1. **성격의 기본 틀** 파악
2. **적합한 직업군** 예측
3. **인생의 과제와 목표** 설정
4. **대운의 길흉** 판단 (격국을 돕는 대운 = 길, 해치는 대운 = 흉)

## 주의사항

격국은 **절대적 운명**이 아닙니다:

- 같은 격국이라도 **용신**에 따라 길흉이 다릅니다
- **대운**과 **세운**의 영향을 받습니다
- **후천적 노력**으로 운명을 개선할 수 있습니다

## 마치며

격국은 사주팔자 분석의 출발점입니다. 자신의 격국을 알면:

- 나의 본성과 재능을 이해할 수 있습니다
- 적합한 인생 방향을 찾을 수 있습니다
- 언제 기회가 오는지 알 수 있습니다
