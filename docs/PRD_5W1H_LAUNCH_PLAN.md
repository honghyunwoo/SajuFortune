## 🎯 5W1H 분석

### 1. WHAT (무엇을)
**제품**: SajuFortune - AI 기반 사주 운세 분석 서비스

**핵심 기능**:
1. **사주팔자 계산** - 생년월일시 기반 정확한 사주 도출
2. **격국 분석** - 8대 정격 + 특수격 판별
3. **대운 분석** - 10년 단위 운세 흐름 (80년 타임라인)
4. **십이운성** - 12단계 생명 에너지 분석
5. **일일운세** - 오늘의 운세 제공
6. **PDF 다운로드** - 분석 결과 저장
7. **궁합 분석** - 두 사람의 궁합 확인

**차별점**:
- 무료 서비스 (광고 기반)
- 전문 명리학 이론 적용 (자평진전, 적천수 기반)
- 확신도 명시 (불확실성 투명 공개)
- 모바일 최적화 UI

### 6. HOW (어떻게)

#### Step 1: 로컬 환경 점검
```bash
cd /home/user/SajuFortune
npm install
npm run build
npm run dev  # 로컬 테스트
```

#### Step 2: P0 보안 이슈 해결
1. 환경변수 검증 강화 (`server/index.ts`)
2. IP 기반 Rate Limiting 추가

#### Step 3: Neon PostgreSQL 설정
1. https://neon.tech 가입
2. 새 프로젝트 생성
3. Connection string 복사
4. `DATABASE_URL` 환경변수 설정

#### Step 4: Upstash Redis 설정
1. https://upstash.com 가입
2. 새 Redis 인스턴스 생성
3. `REDIS_URL` 환경변수 설정

#### Step 5: Railway 배포
1. https://railway.app 가입
2. GitHub 연동
3. 환경변수 설정:
   ```
   NODE_ENV=production
   DATABASE_URL=<Neon URL>
   REDIS_URL=<Upstash URL>
   SESSION_SECRET=<생성한 시크릿>
   ```
4. 배포 트리거

#### Step 6: 도메인 연결
1. 도메인 구매 (가비아, 호스팅KR 등)
2. Railway에서 Custom Domain 설정
3. DNS 레코드 추가

#### Step 7: Google AdSense
1. https://adsense.google.com 가입
2. 사이트 등록
3. 광고 코드 삽입
4. 승인 대기 (1-2주 소요)

#### Step 8: SEO 설정
1. 메타 태그 최적화
2. sitemap.xml 확인
3. Google Search Console 등록
4. "2026년 운세", "신년운세" 키워드 최적화
